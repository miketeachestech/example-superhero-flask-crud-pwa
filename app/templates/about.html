About SuperApp

This website is a simple example of how web applications work using Python and Flask. It demonstrates how websites can store and manage information using a database (SQLite), allowing users to add, edit, and delete superheroes.

Each superhero is stored with important details like their real name, alias, universe, and image, and they can have connected to them. This is an example of a one-to-many relationships in databases, where one superhero can have many powers. Flask, a lightweight web framework, makes it easy to handle user requests, interact with the database, and display dynamic web pages.

Beyond basic functionality, this project introduces key real-world programming concepts like form validation to prevent errors when adding data, file uploads for superhero images, and security measures like CSRF protection to prevent unauthorized form submissions. It also utilizes Bootstrap, a popular CSS framework, to enhance the user experience and make the website both functional and (somewhat) visually appealing.
